# Queue API

> Queue API for the queue Application which is a savings platform for fait & cryptocurrency

## Usage

Rename ".env.example" to ".env" and update the values/settings to your own

## Install Dependencies

Run

```
npm i
```

or

```
npm install
```

## Run Application

```
# Run in dev mode
npm run start:dev

# Run in production mode
npm run start
```
